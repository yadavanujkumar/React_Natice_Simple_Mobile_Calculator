# Calculator App (React Native + Expo)

This is an offline mobile calculator app with scientific functions, EMI calculator, currency converter, and more.

## Features
- Dark/light theme
- History tracking
- BMI calculator
- Tip calculator
- EMI calculator
- Works offline

```bash
npm install
npx expo start
